# Validador de CPF v.1.0.1

Essa biblioteca tem como finalidade validar se um número de CPF é verdadeiro ou falso, retornando **true** ou **false**.

## Como instalar:

```sh
$ npm i cpf-valid
```

## Como utilizar:

```node
> const cpf = require("cpf-valid");
> console.log(cpf(78566133862))
> // returns "true"
```
